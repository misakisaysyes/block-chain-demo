# block-chain-demo

## 运行

* 在本地启动ganache-cli

* 在remix中编写智能合约并编译部署到ganache-cli启用到测试链上

* 部署完后拿到**测试链地址**，**合约地址**，**合约abi**，将配置写到config.js中

* 在package.json文件下运行npm start
